# Thunderstorm Development Index — Meteo 473

## Project Description
This project develops an interactive Thunderstorm Development Index using HRRR model data to assess environments favorable for convective storm development.

## Features
- Interactive plot viewer with forecast hour navigation
- Composite index based on atmospheric variables (CAPE, CIN, moisture, etc.)
- Automated image generation using Python
